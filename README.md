# JavaTextbookCode

This repository holds student-facing Java code referenced by the textbook.

Current layout:

- `chXX-chapter-slug/`: one top-level folder per chapter
- `chXX-chapter-slug/examples/`: Java examples shown directly in the text
- `chXX-chapter-slug/assets/`: reusable student-facing assets referenced by examples or problems
- `chXX-chapter-slug/problems/`: expected output reference files for each textbook problem, containing output examples only

Current chapter folders:

- `ch03-control-structures-and-syntax/`
- `ch07-methods-in-java/`
- `ch15-inheritance-and-composition/`
- `ch17-gui-basics/`
