# Chapter 3

This chapter directory contains the Java code examples that appear in the text.

- `problems/`: expected-output reference files for each textbook problem
- `examples/output/`: `System.out.print`, `println`, and `printf` examples
- `examples/input/`: `Scanner` examples
- `examples/formatting/`: integer formatting and precision examples
- `examples/loops/`: `for`, `while`, and `do-while` demonstration code
- `examples/math/`: `Math` constant examples
- `examples/ascii/`: casting, validation, and character-array examples
- `examples/validation/`: reusable validation pattern example
- `examples/random/`: `Random` and `SecureRandom` examples
