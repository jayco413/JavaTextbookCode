# Chapter 17 GUI Basics

This folder holds the Java code snippets that the textbook explicitly provides in chapter 17.

- `JavaFXWindow.java`: reusable abstract JavaFX window bootstrap class
- `HelloWorldWindow.java`: concrete subclass used by the graphical hello world exercise
- The problem 1 implementation uses the local `earth.png` asset stored under `../assets/earth.png`.

The chapter's FXML layouts are mostly described in prose rather than printed in full code listings, so the rest of the chapter's student-facing references live under `../problems/` as expected-output guides.
